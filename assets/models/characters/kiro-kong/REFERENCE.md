# Kiro Kong Model Reference
## PRIMAL BREAKER

**Character ID:** KIRO_KONG  
**Species:** Augmented Ape-Kin  
**Height:** 6.5 feet hunched / 8.0 feet standing (1.98m / 2.44m)

## Visual Description
Kiro Kong is a massive augmented gorilla with impact-dampening stone armor plates. He has mechanical enhancements and creates powerful ground-slam shockwaves.

## Key Dimensions
- Height: 6.5 units (hunched posture)
- Standing height: 8.0 units
- Arm span: 9.0 units (massive reach)
- Arms: 4.0 units each
- Legs: 3.0 units
- Torso: 2.5 units (barrel-chested)

## Material Breakdown
- **Fur:** Realistic gorilla fur with grooming details
- **Stone Armor:** Weathered rock PBR with moss/cracks
- **Mechanical Parts:** Metallic pistons with rust and wear
- **Eyes:** Amber emissive glow

## Color Palette
- Fur: #3d2817 (dark brown)
- Stone Armor: #808080 (stone gray) + #4a7c59 (moss green)
- Mechanical Parts: #666666 (steel) + #8b4513 (rust)
- Eyes: #ffaa00 (amber)

## Special Features
- **Impact-Dampening Armor:** Stone plates crack when damaged
- **Augmented Arms:** Hydraulic pistons visible at joints
- **Ground Slam VFX:** Dust/debris burst, shockwave ring
- **Damage States:** Progressive cracking textures that heal over time

## Animation Notes
- Idle: Chest-beating, intimidation stance
- Walk: Knuckle-walk (quadrupedal)
- Run: Full gorilla charge
- Jump: Powerful leg extension
- Attack: Fist slam, creates shockwave
- Special: Full-power ground slam (screen shake + debris)

## Damage State Textures
Create 3 damage variations:
1. **Pristine:** Clean stone armor (0% damage)
2. **Cracked:** Minor cracks (50% damage)
3. **Shattered:** Major cracks (80% damage)
