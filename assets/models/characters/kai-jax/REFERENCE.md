# Kai-Jax Model Reference
## THE MEMORY KING

**Character ID:** KAI_JAX  
**Species:** Star-Slime Chimera (Hedgehog-Lupine Fusion)  
**Height:** 3.5 feet (1.07 meters)

## Visual Description
Kai-Jax is a compact, Kirby-esque character with a spherical charcoal body containing swirling internal nebulae. He has 3 distinct memory tails and jagged electric quills.

## Key Dimensions
- Body diameter: 1.0 unit
- Head: 0.8x body width
- Arms: 0.4 units
- Legs: 0.5 units
- Tail length: 1.5 units each

## Tail Specifications
1. **Tail 1 (Velocity/Jax)** - Liquid ink with hard-light echo trail
2. **Tail 2 (Shielding/Kai)** - Thicker, semi-solid with web-tether capability
3. **Tail 3 (Father's Strand)** - Translucent, ghostly appearance

Each tail requires 10 bones for full animation range.

## Material Breakdown
- **Fur:** Dual-layer shader (opaque charcoal + translucent nebula)
- **Quills:** Metallic with emissive blue tips
- **Eyes:** Neon-gold emissive with slit pupils
- **Tails:** Liquid (T1), matte rubber (T2), ghostly transparency (T3)

- Jump: Tails whip upward for momentum
- Attack: Tail 1 lashes forward with echo trails
