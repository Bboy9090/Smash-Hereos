# Umbra-Flux Model Reference
## VELOCITY WRAITH

**Character ID:** UMBRA_FLUX  
**Species:** Celestial Lupine (Star-Wolf/Lynx Hybrid)  
**Height:** 4.2 feet (1.28 meters)

## Visual Description
Umbra-Flux is a sleek, streamlined wolf with matte-white metallic fur and 5 iridescent elemental tails. He has dual-pupil eyes and crystalline quill-blades on his back.

## Key Dimensions
- Height: 4.2 units
- Body length: 3.5 units (horizontal stance)
- Legs: 2.0 units
- Tail length: 7.6 units each

## Tail Specifications
1. **Tail 1 (Blue/Sonic)** - Light-speed after-images
2. **Tail 2 (Red/Shadow)** - Teleportation anchor particles
3. **Tail 3 (Cyan/Silver)** - Psychokinetic object trails
4. **Tail 4 (Violet/Hypersonic)** - Time distortion blur
5. **Tail 5 (Gold/Celestial)** - Reality anchor shimmer

Each tail requires 10 bones and unique shader effects.

## Material Breakdown
- **Fur:** Matte-white metallic with anisotropic highlights
- **Quills:** Crystal shader with refraction
- **Eyes:** Dual-pupil (left sees past, right sees future)
- **Skates:** Hard-light emissive with rainbow trail

## Color Palette
- Base: #f0f0f0 (matte-white)
- Tail 1: #3399ff (blue)
- Tail 2: #ff3333 (red)
- Tail 3: #00ffff (cyan)
- Tail 4: #aa00ff (violet)
- Tail 5: #ffd700 (gold)
- Eyes: Chromatic aberration effect

## Special Features
- **Dual-Pupil Eyes:** Layered vision with chromatic aberration
- **Quill-Blades:** Vibrate at hypersonic frequency (motion blur)
- **Echo Skates:** Hover 2cm above ground, rainbow particle trail

## Animation Notes
- Idle: Quills vibrate subtly, tails sway independently
- Run: Echo skates activate, light trails appear
- Jump: Time distortion blur effect on Tail 4
- Attack: Tails whip in sequence, each with unique VFX
