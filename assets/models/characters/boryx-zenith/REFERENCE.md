# Boryx Zenith Model Reference
## GUARDIAN KING

**Character ID:** BORYX_ZENITH  
**Species:** Draconic Ursine (Dragon-Bear)  
**Height:** 7.0 feet (2.13 meters)

## Visual Description
Boryx is a massive heavyweight fighter with a mix of bear fur and dragon scales. He has a glowing chaos star embedded in his chest and wears a heavy aero-cape.

## Key Dimensions
- Height: 7.0 units
- Shoulder width: 3.0 units (broad)
- Arms: 2.5 units (thick)
- Legs: 3.2 units (tree trunks)
- Weight class: 110 (heaviest)

## Material Breakdown
- **Fur:** Realistic bear fur with subsurface scattering
- **Scales:** Dragon scales on shoulders/forearms/chest (metallic PBR)
- **Cape:** Heavy cloth simulation with bronze trim
- **Source Star:** Emissive amber core with particle burst

## Color Palette
- Fur: #5c4033 (Bronx brown)
- Scales: #cd7f32 (bronze)
- Cape: #8b4513 (brown) with #cd7f32 (bronze trim)
- Source Star: #ffbf00 (amber)
- Eyes: #ff6600 (amber-red)

## Special Features
- **Chaos-Infused Source Star:** Chest-mounted, pulsates with power
- **Aero-Cape:** Flares dramatically during attacks
- **Boxing Stance:** Wrapped hands with brass knuckles
- **Mixed Texture:** 50/50 fur and scale coverage

## Animation Notes
- Idle: Heavy breathing, cape sways
- Walk: Ground shakes slightly (screen shake)
- Run: Quadrupedal charging stance
- Attack: Cape flares, source star pulses
- Special: Source star emits particle burst
