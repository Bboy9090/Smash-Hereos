# Sentinel Vox Model Reference
## STAR-FORCE KITSUNE

**Character ID:** SENTINEL_VOX  
**Species:** Saiyan-Kitsune (Buff Fox-Alien)  
**Height:** 5.5 feet (1.68 meters)

## Visual Description
Sentinel Vox is a muscular fox pilot with mechanical energy tail-blades and a tactical Star-Force flight jacket. He has glowing blue whisker-scars and a retractable HUD visor.

## Key Dimensions
- Height: 5.5 units
- Upper body: Muscular (1.2x normal proportions)
- Arms: 2.2 units
- Legs: 2.8 units
- Tail-blades: 3.0 units (extended)

## Tail Specifications
- **Mechanical Tail-Blades:** 2-9 configuration (starts 2, extends to 9)
- Segmented metallic design with blue emissive edges
- Each blade: 8 bones for articulation

## Material Breakdown
- **Fur:** Fox fur shader (orange/white)
- **Jacket:** Leather PBR with cloth simulation
- **Tail-Blades:** Metallic with emissive blue edges
- **Visor:** Glass shader with HUD overlay texture

## Color Palette
- Fur: #ff6600 (orange) + #ffffff (white chest)
- Jacket: #0066cc (tactical blue) + #ff6600 (trim)
- Tail-blades: #333333 (metal) + #3399ff (blue glow)
- Visor: #ffaa00 (orange tint)
- Whisker-scars: #3399ff (blue emissive)

## Special Features
- **Tactical Flight Jacket:** Star-Force insignia patches
- **Retractable Visor:** HUD overlay with tactical info
- **Glowing Whisker-Scars:** Battle damage aesthetic (6 per side)
- **Buff Build:** Muscular upper body, pilot stance

## Animation Notes
- Idle: Arms crossed, visor flickers
- Run: Tail-blades extend, pilot stance
- Jump: Jacket flutters, visor activates
- Attack: Tail-blades spin/slash, whiskers glow brighter
- Special: All 9 tail-blades extend simultaneously
