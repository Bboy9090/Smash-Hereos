# Lunara Solis Model Reference
## ORACLE SENTINEL

**Character ID:** LUNARA_SOLIS  
**Species:** Celestial Feline-Avian Hybrid (9-tailed Kitsune)  
**Height:** 5.8 feet (1.77 meters)

## Visual Description
Lunara is an elegant kitsune with liquid starlight fur that shifts between gold and silver. She has 9 flowing tails and wears aurora borealis robes with an ornate holographic crown.

## Key Dimensions
- Height: 5.8 units
- Torso: 1.5 units
- Legs: 2.8 units (digitigrade stance)
- Arms: 2.0 units
- Tail length: 11.6 units each (2x height)

## Tail Specifications
- **5 Gold Solar Tails:** Fire particle emitters, gold gradient
- **4 Silver Lunar Tails:** Shimmer effects, silver gradient
- **Merged Form:** All tails combine into single Titan Tail (ultimate)

Each tail requires 12 bones for fluid animation.

## Material Breakdown
- **Fur:** Iridescent shader (gold/silver shift based on lighting)
- **Robes:** Cloth simulation with aurora gradient texture
- **Crown:** Holographic shader → solid metal at 100% Resonance
- **Eyes:** Left gold (thermal), right silver (probability threads)

## Color Palette
- Fur: #ffd700 (gold) ↔ #c0c0c0 (silver) - shifts with lighting
- Robes: Aurora gradient (#ff6ec7 → #00ffa3 → #3399ff)
- Crown: #ffffff (holographic) → #ffd700 (solid gold)
- Eyes: Left #ffaa00, Right #aaddff

## Special Features
- **Crown of Infinite Stars:** Appears at 60%+ Resonance, 12 floating points
- **Aether Silk Robes:** Leave healing particle trails
- **Duality Eyes:** Each eye has unique shader effect

## Animation Notes
- Idle: Tails flow gently, robes ripple
- Run: Solar tails trail fire, lunar tails create after-images
- Jump: Robes billow upward, crown materializes
- Special: Tails merge into Titan Tail, crown becomes solid
